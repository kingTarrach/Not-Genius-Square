# Genius Square
PA 9
Game is a 6x6 grid
There are 7 blocking circles that can be placed anywhere on the board
there are also 9 game pieces as follows:
  1x1
  1x2
  1x3
  1x4
  2x2
  1x2 2x1 (square missing one)
  1x2 3x1 (L piece)
  3x1 2x1 (tetris T)
  2x1 1x2 (1x2 stacked on 1x2 offset by one)
